# Phase 3 — Data Processing

**Weeks 9–12 &nbsp;|&nbsp; Milestone: M3**

> Raw data is never analysis-ready. Phase 3 is about understanding what you have, fixing what's broken, and producing a clean dataset you can trust.

---

## Phase Goal

By the end of Week 12, you will have:

- A clean, structured dataset saved in `data/processed/`
- A working cleaning script (`scripts/02_clean.py`)
- Documented decisions about how you handled messy data
- Completed Milestone 3 submission

---

## Weekly Breakdown

| Week | Focus | Task |
|------|-------|------|
| [Week 9](week-09/) | Data Profiling | Understand your raw data: shape, types, nulls, duplicates. |
| [Week 10](week-10/) | Handling Missing Values | Drop, fill, or flag — and document your decisions. |
| [Week 11](week-11/) | Data Types & Schema | Ensure dates are dates, numbers are numbers. Define your schema. |
| [Week 12](week-12/) | Cleaning Script | Finalize `02_clean.py`. Save to `data/processed/`. Submit M3. |

---

## Milestone 3 — Clean Dataset

**Due:** End of Week 12
