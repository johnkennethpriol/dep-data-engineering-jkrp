# Phase 6 — Deployment

**Weeks 21–24 &nbsp;|&nbsp; Milestone: M6**

> The final step. A project that lives only on your laptop isn't a portfolio piece. Phase 6 puts your work on the internet, permanently and publicly.

---

## Phase Goal

By the end of Week 24, you will have:

- A working `scripts/03_analyze.py` that generates `dashboard/index.html`
- A live dashboard deployed to GitHub Pages with a public URL
- A GitHub Actions workflow that auto-deploys on every push to `main`
- Completed Milestone 6 submission — and a project you can share anywhere
